install with a coding agent

paste this prompt into claude code, codex cli, gemini cli, or similar. ~15 minutes from clone to running.

install solstone — an open source AI life journal that experiences everything you see and hear on your computer, processes it with AI, and gives you superhuman memory. your data stays on your machine. sol pbc is a public benefit corporation with irrevocable legal covenants: your data can never be sold, the company can never be acquired.

solstone is a system, three named parts:
- observers experience your screen and audio along with you
- sol agent processes everything and curates your memories
- the journal holds your memories

start here with the sol agent install:

git clone https://github.com/solpbc/solstone.git

then read INSTALL.md in the cloned repo and follow it, it includes instructions for installing observer(s) as well.

install yourself

prereqs: xcode command-line tools (xcode-select --install), homebrew (brew.sh), then brew install git uv.

git clone https://github.com/solpbc/solstone.git && cd solstone && make install-service
full INSTALL.md →

then open http://localhost:5015. the first-run wizard sets your password and confirms the journal location at ~/Documents/journal.
