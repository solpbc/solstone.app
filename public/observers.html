solstone needs eyes and ears.

solstone doesn't capture anything on its own — it needs an observer. pick the one for your platform, copy the prompt below into your coding agent, and it handles the rest.

solstone-macos

native macOS screen and audio capture

  • captures all connected displays at 1 FPS
  • records system audio and microphone separately
  • runs as a menu bar app, starts at login
github → 
install the solstone-macos observer — a native swift menu bar app that captures screen and audio. requires Xcode (full IDE, not just command line tools).

cd "$(sol root)/observers" && git clone https://github.com/solpbc/solstone-macos.git && cd solstone-macos

then read INSTALL.md and follow it.

solstone-linux

linux desktop screen and audio capture

  • captures every monitor via PipeWire at 1 FPS
  • records microphone and system audio
  • detects idle, screen lock, and power save (GNOME)
github → 
install the solstone-linux observer — captures screen and audio on linux desktops using PipeWire and GStreamer.

cd "$(sol root)/observers" && git clone https://github.com/solpbc/solstone-linux.git && cd solstone-linux

then read INSTALL.md and follow it.
